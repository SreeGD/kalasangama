# Print files

**`kala-sangama-posters-print.pdf`** — press-ready, 2 pages (English + Kannada).

| Spec | Value |
|------|-------|
| Trim size | **300 × 400 mm** (30 × 40 cm, 3:4) |
| Bleed | **3 mm** all sides |
| Page (trim + bleed) | 306 × 406 mm |
| Crop marks | Yes (corner trim marks) |
| Resolution | ~300 DPI |
| Colour | RGB |

## Give this to the printer
Hand over the PDF as-is. Background art runs into the 3 mm bleed, and crop marks show
the 300 × 400 mm trim. Ask them to **print actual size / 100 % (no "fit to page")** so the
bleed and marks stay correct.

## Notes
- **Colour:** exported in RGB. Most digital/print shops convert to CMYK fine; if your
  press requires a CMYK PDF, ask them to convert, or tell me and I'll regenerate.
- **Other sizes:** the design is 3:4, so it scales cleanly to 45 × 60 cm, 18 × 24 in, etc.
  A3 (297 × 420 mm) is *not* 3:4 — it would add side margins. Tell me the exact size and
  I'll regenerate.
- **Regenerating:** the source is `assets/_poster-build.html`. Edit text/dates there, then
  re-render — ask me and I'll rebuild the PNGs and this PDF.
