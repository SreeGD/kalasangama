# Local-mode images

In `INPUT_MODE=local`, put the artwork photos here and reference each filename in the
`image` column of `../submissions.csv`. Supported: `.jpg .jpeg .png .webp .gif`.

If you collected submissions through the **Artwork Submission** Google Form, you can
either:
- download the Drive folder of uploads into this folder and build the CSV, **or**
- switch to `INPUT_MODE=drive` to read the Sheet + Drive directly (see the main README).
